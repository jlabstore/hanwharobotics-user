HCT-1200

Dymato Automation

Description
The HCT-1200 table is specifically designed for machine tending solutions with collaborative robots. All the cobot components, such as the controller box and teach pendant, are integrated into the table, making the HCT-1200 movable with a pallet truck. Additionally, the HCT-1200 includes Plug&Play software, compatible with all HCR (Hanwha Collaborative Robots).

Specifications

Drawer for tools and manuals

Storage for 8 different part trays

Color

Steel parts powder coated in Jet Black RAL9005

Software

Custom made software based on Rodi-X for Pick&Place solutions

* The HCT-1200 standard includes a mounting support for the HCR-12A, a mounting support for the teach pendant, and a stainless steel tray with 60 positions of 100x100mm.
